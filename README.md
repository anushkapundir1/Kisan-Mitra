## Overview
Kisan AI Mitra is a Python-based farmer assistance system aimed at
providing basic agricultural guidance through structured logic and
AI-assisted responses.

The project focuses on helping users (farmers) get simple,
understandable recommendations related to crops, farming practices,
and common agricultural queries.


## Key Features
- Simple and farmer-friendly input flow
- Rule-based agricultural guidance
- AI-assisted response generation
- Designed for real-world rural use cases


## Tech Stack
- Python
- AI / LLM APIs
- Rule-based logic


## Learning Outcomes
- Designing AI systems for non-technical users
- Combining rule-based logic with AI assistance
- Structuring domain-specific AI applications
- Understanding real-world problem constraints


## Future Improvements
- Local language support
- Web or mobile interface
- Integration with real agricultural datasets
