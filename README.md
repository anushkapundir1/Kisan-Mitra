## Overview
Kisan AI Mitra is a Python-based farmer assistance system aimed at
providing basic agricultural guidance through structured logic and
AI-assisted responses.

The project focuses on helping users (farmers) get simple,
understandable recommendations related to crops, farming practices,
and common agricultural queries.


## Key Features
- Upload and analyze crop and soil images
- AI-based disease detection
- Fertilizer and crop-care recommendations like temperature, climate, soil conditions
- Simple and farmer-friendly web interface

---

## Tech Stack
- Python
- Django
- Gemini Vision API
- HTML, CSS

---

## How It Works
1. User uploads crop or soil images
2. Images are sent to the Gemini Vision API
3. AI analyzes visual patterns for disease indicators
4. Results are displayed in a simple web interface
